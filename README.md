# 🤖 Automation Portfolio

Python automation projects by Wann — AI Automation Specialist in training.

---

## Projects

### 1. 📈 Price Tracker
Scrape harga dari website secara otomatis dan kirim notifikasi ke Telegram kalau harga turun.
- **Tech:** Python, BeautifulSoup, Requests
- **Folder:** `price-tracker/`

### 2. 🧹 CSV Cleaner
Script otomatis untuk membersihkan data kotor dari file CSV — hapus duplikat, standarisasi format, isi kolom kosong.
- **Tech:** Python, Pandas
- **Folder:** `data_cleaning/`

### 3. 📊 Auto Report Generator
Baca data penjualan dari CSV, hitung summary otomatis, kirim laporan harian ke Telegram.
- **Tech:** Python, Pandas, Telegram Bot API
- **Folder:** `report-generator/`

### 4. 📬 Multi-platform Notifier
Nerima data dari form, otomatis kirim notifikasi ke Telegram dan Email sekaligus.
- **Tech:** Python, Flask, Telegram Bot API, SMTP Gmail
- **Folder:** `notifier/`

### 5. 🤝 Lead Auto-responder
Baca lead baru dari Airtable, kirim email balasan otomatis yang dipersonalisasi, update status di Airtable.
- **Tech:** Python, Airtable API, SMTP Gmail
- **Folder:** `lead-autoresponder/`
